### Week 3 Achievements:

* Successfully completed hands-on labs for **AWS Identity and Access Management (IAM)** and learned how to:
  * Manage IAM Users and IAM Groups.
  * Grant permissions using IAM Policies.
  * Assign IAM Roles following the **Principle of Least Privilege**.

* Successfully deployed applications on **Amazon EC2** running both Windows and Linux operating systems.

* Created **Amazon EBS Snapshots** to back up EC2 volumes and gained an understanding of backup and recovery procedures.

* Completed hands-on labs for **Amazon EC2 Auto Scaling**, including:
  * Configuring Auto Scaling Groups.
  * Setting up an **Application Load Balancer (ALB)**.
  * Understanding automatic scaling based on workload demand.
  * Learning backup and recovery strategies to improve system availability.

* Configured **Amazon CloudWatch** and **CloudWatch Alarms** to:
  * Monitor AWS resources.
  * Collect operational metrics.
  * Generate alerts when predefined thresholds are exceeded.

* Successfully deployed a **WordPress** website on **Amazon Lightsail**, gaining practical experience with simplified cloud application deployment.
