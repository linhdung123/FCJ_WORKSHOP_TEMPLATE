### Week 1 Achievements:

* Became familiar with the internship environment, met the members of the First Cloud Journey (FCJ) program, and understood the rules and regulations to be followed throughout the internship.
* Successfully registered an AWS account and completed the required tasks to receive AWS Free Credits, preparing the environment for future hands-on labs.
* Gained a solid understanding of Amazon VPC, including its concepts, purpose, and role in building secure cloud network infrastructure, as well as the differences and key features of various AWS compute services.
* Developed a deeper understanding of essential AWS networking components such as Subnets, Route Tables, Internet Gateway (IGW), NAT Gateway, and Security Groups, including how these components work together to build a secure and reliable network architecture.
* Successfully completed hands-on Amazon VPC labs, including creating a VPC, configuring Subnets, and setting up basic networking components, thereby strengthening both theoretical knowledge and practical AWS skills.
